公理表

  • I1 任意の相異なる 2点に対して, この 2点を通る直線がただ 1つ存在する.
  • I2 任意の直線に対して, この直線が通る点が 2つ (以上) 存在する.
  • I3 ある 3点が存在して, どんな直線も「これら 3点すべてを通る」ということはない.
  • B1 点 B が点 A と点 C の間にあるならば, 点 A, B, C は一直線上の相異なる 3 点であり, 点 B は点 C と点 A の間にある.
  • B2 任意の相異なる2点A,Bに対して,「点Bは点Aと点Cの間にある」となる点Cが 存在する
  • B3 一直線上の相異なる 3 点に対して, このうちのちょうど 1 点のみが残る 2 点の間にある.
  • B4 「Pasch の公理」. 一直線上にない 3 点 A, B, C と, この 3 点のいずれをも通らない直線 l があるとする. 直線 l が「点 A と点 B の間にある点を通る」ならば, 「点 A と点 C の間にある点 を通る」または「点 B と点 C の間にある点を通る」のどちらか一方が成り立つ.
  • C1 任意の線分 AB と, 点 C を端点とする半直線 r に対して, 線分 AB と線分 CD が等しい, となる点 D が半直線 r 上にただ 1 つ存在する.
  • C2
        反射律 (reflexive law): 線分 AB と線分 AB が等しい.
        対称律 (symmetric law): 線分 AB と線分 CD が等しいならば, 線分 CD と線分 AB が等しい.
        (3) 推移律 (transitive law): 線分 AB と線分 CD が等しいかつ線分 CD と線分 EF が等しいなら ば, 線分 AB と線分 EF が等しい.
  • Ｃ3 点Bが点Aと点Cの間にあり,点Eが点Dと点Fの間にあるとする. 線分ABと線 分 DE が等しいかつ線分 BC と線分 EF が等しいならば, 線分 AC と線分 DF が等しい.
